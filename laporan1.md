## Tujuan Praktikum
1. Memahami fungsi dan kegunaan aplikasi Wireshark dalam analisis jaringan komputer.
2. Mengetahui cara menginstall aplikasi Wireshark beserta komponen pendukungnya seperti Npcap dan USBPcap.
3. Memahami langkah-langkah dasar dalam menyiapkan tools yang digunakan pada praktikum jaringan komputer.
4. Mempersiapkan lingkungan praktikum agar dapat digunakan untuk menganalisis paket data jaringan pada pertemuan selanjutnya.

## Langkah Percobaan
1. Pertama saya membuka modul praktikum jaringan komputer untuk melihat tools yang perlu diinstal sebelum melakukan praktikum.
2. Selanjutnya saya membuka website resmi Wireshark melalui browser untuk mengunduh aplikasi Wireshark.
3. Setelah itu saya masuk ke halaman download Wireshark dan memilih installer yang sesuai dengan sistem operasi yang digunakan.
4. Kemudian saya melakukan proses pengunduhan file installer Wireshark hingga selesai.
5. Setelah proses download selesai, file installer Wireshark tersimpan pada komputer dan siap untuk dijalankan.
6. Selanjutnya saya menjalankan file installer Wireshark sehingga muncul tampilan awal instalasi, kemudian saya menekan tombol *Next* untuk melanjutkan proses instalasi.
7. Setelah itu muncul halaman *License Agreement*, pada tahap ini saya membaca perjanjian lisensi kemudian menyetujuinya untuk melanjutkan proses instalasi.
8. Pada tahap berikutnya muncul informasi mengenai *Wireshark Certified Network Analyst (WCNA), kemudian saya menekan tombol **Next* untuk melanjutkan instalasi.
9. Selanjutnya muncul halaman *Choose Components* yang berisi pilihan komponen yang akan diinstal. Saya membiarkan pilihan komponen pada pengaturan default kemudian menekan tombol *Next*.
10. Pada tahap berikutnya muncul halaman *Additional Tasks* yang berisi beberapa pilihan tambahan seperti pembuatan shortcut aplikasi. Saya membiarkan pengaturan tetap default kemudian menekan tombol *Next*.
11. Setelah itu muncul halaman pemilihan lokasi instalasi Wireshark pada komputer. Saya menggunakan lokasi instalasi default yang berada pada folder *Program Files* kemudian menekan tombol *Next*.
12. Selanjutnya sistem melakukan instalasi *Npcap*, yaitu driver yang digunakan untuk menangkap paket data jaringan. Saya mengikuti proses instalasi hingga selesai.
13. Pada tahap terakhir sistem melakukan instalasi *USBPcap* sebagai fitur tambahan untuk menangkap lalu lintas data dari perangkat USB. Setelah proses instalasi selesai, aplikasi Wireshark berhasil terpasang dan siap digunakan.
