## Tujuan Praktikum
1. Memahami cara menggunakan aplikasi Wireshark untuk melakukan packet capturing pada jaringan.
2. Mengetahui cara memilih interface jaringan yang digunakan untuk menangkap paket data.
3. Menganalisis paket data yang lewat di jaringan menggunakan filter pada Wireshark.
4. Mengidentifikasi komunikasi HTTP request dan response yang terjadi antara client dan server.
5. Memahami struktur paket jaringan yang ditampilkan oleh Wireshark seperti Ethernet, IP, TCP, dan HTTP.

## Langkah Percobaan
1. Menginstall dan membuka aplikasi Wireshark pada komputer.
2. Pada halaman awal Wireshark, memilih interface jaringan Wi-Fi yang sedang digunakan untuk koneksi internet.
3. Memulai proses packet capturing dengan menekan tombol Start.
4. Mengakses sebuah website melalui browser untuk menghasilkan trafik jaringan.
5. Menggunakan display filter http pada kolom filter untuk menampilkan hanya paket HTTP.
6. Memilih salah satu paket HTTP yang muncul pada daftar paket untuk dianalisis lebih lanjut.
7. Melihat detail paket pada bagian Packet Details dan Packet Bytes untuk memahami struktur paket jaringan.
8. Menghentikan proses capture setelah data yang diperlukan sudah didapatkan.
